¿Qué es GIT?

Git es un sistema de control de versiones de código fuente gratuito y de código abierto. Fue creado en 2005 por Linus Torvalds, el creador del kernel de Linux, y es ampliamente utilizado por desarrolladores de software para colaborar en proyectos de software y rastrear cambios en el código fuente a lo largo del tiempo.

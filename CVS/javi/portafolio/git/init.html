Init

El comando "init" es utilizado en Git para inicializar un nuevo repositorio Git vacío en un directorio existente.

Cuando se ejecuta este comando en un directorio específico, Git crea una nueva subcarpeta oculta llamada ".git" dentro del directorio seleccionado. Esta subcarpeta es donde Git almacena todos los metadatos y el historial de versiones de los archivos del repositorio.

Después de ejecutar el comando "init", puedes empezar a agregar archivos al repositorio Git, crear ramas, hacer commits, y realizar otras operaciones que te permiten controlar la versión y la gestión de tus archivos.

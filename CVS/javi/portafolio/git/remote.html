Remote

El comando "git remote" es una herramienta en Git que te permite ver, agregar y eliminar conexiones a repositorios remotos.

Un repositorio remoto es una copia de tu repositorio local en un servidor remoto, como GitHub, Bitbucket o GitLab. Cuando trabajas en un proyecto con colaboradores o quieres hacer una copia de tu código en un lugar seguro y accesible desde cualquier lugar, puedes crear un repositorio remoto.

  • Agregar una nueva conexión remota: git remote add <nombre-remoto>> <url-remoto>
  • Cambiar el nombre de una conexión remota: git remote rename <nombre-actual> <nuevo-nombre>
  • Eliminar una conexión remota: git remote remove <nombre-remoto>
  • Ver información detallada sobre una conexión remota: git remote show <nombre-remoto>

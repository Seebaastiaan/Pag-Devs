Checkout

El comando "git checkout" es una herramienta en Git que te permite cambiar entre ramas, crear nuevas ramas, cambiar a una confirmación anterior y restaurar archivos eliminados.

Algunos de los usos más comunes del comando "git checkout" incluyen:

  • Cambiar a una rama diferente: git checkout <nombre-de-la-rama>
  • Crear y cambiar a una nueva rama: git checkout -b <nombre-de-la-rama>
  • Cambiar a una confirmación anterior: git checkout <hash-de-la-confirmación>
  • Restaurar un archivo eliminado: git checkout -- <nombre-del-archivo>
  • Restaurar un archivo eliminado en una confirmación anterior: git checkout <hash-de-la-confirmación> -- <nombre-del-archivo>

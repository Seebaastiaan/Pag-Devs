Branch

El comando "git branch" es una herramienta en Git que te permite ver, crear, borrar y renombrar ramas en un repositorio Git.

Una rama en Git es una línea de desarrollo independiente que te permite trabajar en características o mejoras en tu código sin afectar directamente la rama principal o la rama "master". Las ramas son útiles porque te permiten mantener diferentes versiones de tu código y fusionar cambios de diferentes ramas para crear una versión final.

Cuando ejecutas el comando "git branch" en la línea de comandos, Git te muestra una lista de todas las ramas en el repositorio, con un asterisco que indica la rama actual.
